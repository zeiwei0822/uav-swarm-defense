# 無人機機群攻防模擬 — 台灣海峽防衛戰

以 Python 打造的無人機**機群攻防**模擬：對岸機群跨海峽進襲、我方防空攔截，並用 **AI** 識別指揮節點、預測軌跡。專題核心在比較「**傳統防空 vs AI 防空**」，並依真實作戰準則區分兩大攻擊流派，凸顯 AI 斬首戰術的**價值與極限**。

## 🔗 線上展示（手機可看）

**https://zeiwei0822.github.io/uav-swarm-defense/**

選一種陣形、切換「傳統／AI」，看同一場景的對比：
- **無線大群**（有指揮鏈）：AI 認出領機 → 斬首 → **癱瘓整群**，遠勝傳統。
- **光纖精準群**（抗干擾＋抗斬首）：領機被打掉、後機照航跡續突 → **AI 斬首失效**。

## 兩大攻擊流派（依真實作戰準則）

| 流派 | 規模 | 陣形 | 特性 |
|---|---|---|---|
| 🛰️ **無線大群** | 數十～數百架 | 寬幅梯次橫隊 · 重疊錐形 · 向心多弧包圍 | 有 C2 指揮鏈 → **可被斬首**；常帶電戰壓制防空 |
| 🔗 **光纖精準群** | 3–8 架 | 一字蛇形 · 垂直梯次階梯 · 母子跳島 · 扇形向心 | 實體光纖 → **免疫電子干擾**；領機亡、後機沿航跡續突 → **抗斬首** |

## 專題四要點

1. **機群陣型與飛行路徑** — 兩流派共 7 種陣形，各有隱蔽性、通訊、電戰／光纖的取捨。
2. **領機／中繼機失效處理** — 心跳逾時 → 選舉 → 中繼機遞補 → 編隊重組；光纖群為**無縫接替**。
3. **AI 軌跡預測** — LSTM（位移＋航向旋轉對齊）預測機群未來位置供攔截導引，勝卡爾曼 CV 約 **48%**（ADE ≈ 6.7 m）。
4. **AI 找出領機／中繼機** — 樹模型從 9 維飛行行為特徵識別指揮節點（領機 Top-1：RF ≈ **0.87**、規則法 ≈ 0.51）。

## 戰術與對抗

- **指揮節點打擊（斬首）**：AI 認出領機後優先擊殺、打斷指揮鏈，達成傳統做不到的「癱瘓整群」——**僅對無線大群有效**。
- **光纖抗斬首**：光纖群把航線存進每架機、又免疫干擾，斬首任一節點都無法癱瘓整群——示範行為識別（要點4）的**極限**。
- **電子干擾（EW）**：無線群帶電戰載荷壓制防空接戰距離（抽象為 engage 距離縮減）。
- **誘餌／沉默領機**：誘餌衝前方扮假領機、真領機藏身 → AI 多半把火力打在誘餌上（行為識別的另一死穴，可選開啟）。

## 執行（桌面控制台）

需求：Python 3.10、`numpy / matplotlib / scikit-learn / torch / Pillow`。

```bash
# Windows 請先設定 UTF-8 避免 cp950 編碼錯誤
set PYTHONUTF8=1            # PowerShell: $env:PYTHONUTF8='1'

python train.py            # 首次：產生 models/（訓練 LSTM / RF / GBM / MLP，約數分鐘）
python app.py              # 開啟整合控制台（選陣形/防方 → 開戰）
```

控制台操作：上排選**陣形**（兩流派 7 種，換陣型自動帶入合理機數）、**防方（傳統／AI）**、種子 → 按「開戰」。
跑完用底部播放列觀戰（上一/下一重點、播放、0.25～4× 變速、導覽模式自動暫停解說）。
其他：`python run_sim.py`（單場／匯出動畫）、`python analyze.py`（分析圖）。

## 架構

| 模組 | 說明 |
|---|---|
| `core/formations.py` | 7 種陣型庫、兩流派 metadata、隱蔽/通訊/電戰取捨 |
| `core/swarm.py` | 機群階層通訊、失效偵測/選舉/遞補/重組、光纖抗斬首、蛇形沿跡尾隨 |
| `core/defense.py` | 雷達、AI 識別排程、火控策略（傳統最近 / AI 指揮節點打擊）、攔截彈導引、EW 壓制 |
| `core/engine.py` | 攻防主循環、全程記錄器、戰報 |
| `ai/` | LSTM 軌跡預測（`trajectory.py`）、RF/GBM/MLP 角色識別（`identify.py`）、訓練資料生成（`datagen.py`） |
| `viz/tactical2d.py` | 2D 俯視戰術地圖＋即時遙測側欄＋資料驅動分幕旁白（流派感知） |
| `app.py` | tkinter 整合控制台（固定海峽視角） |
| `docs/` | 線上展示網頁（GitHub Pages） |

## 已驗證成果

- **軌跡預測**：LSTM ADE 6.7 m，勝卡爾曼 CV 約 48%（2.5 s 預測時域）。
- **領機 Top-1 識別**（跨 7 陣型）：RF ≈ 0.87、GBM ≈ 0.88、MLP ≈ 0.86、規則法 ≈ 0.51。
- **離線 ≠ 線上**：GBM 離線 Top-1 最高，但在動態模擬中**斬首反而失效**（時序穩定度差）；RF 線上斬首最有效 → **產品採用 RF**。這是「離線指標 ≠ 部署效能」的實證。
- **兩流派對照**：無線群（如重疊錐形）AI 斬首 → 機群癱瘓，遠勝傳統；光纖群（如一字蛇形）AI 斬首失效、突防數與傳統相當。
